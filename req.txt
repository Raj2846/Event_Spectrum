# Event Management System - Complete Setup Guide

## Requirements (requirements.txt)
```

razorpay==1.4.1
django-cors-headers==4.3.1
```

## Installation Steps

### 1. Create Virtual Environment
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. MySQL Database Setup
```sql
CREATE DATABASE event_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'event_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON event_db.* TO 'event_user'@'localhost';
FLUSH PRIVILEGES;
```

### 3. Configure Settings
Update `settings.py` with:
- MySQL credentials
- Razorpay API keys (get from https://dashboard.razorpay.com/app/keys)
- Email configuration (Gmail App Password: https://support.google.com/accounts/answer/185833)

### 4. Run Migrations
```bash
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
```

### 5. Create Media Directories
```bash
mkdir -p media/qr_codes media/event_images
```

### 6. Run Development Server
```bash
python manage.py runserver
```

## Razorpay Setup (Free Test Mode)

1. Sign up at https://razorpay.com/
2. Go to Settings → API Keys
3. Generate Test Keys (free for testing)
4. Add keys to `settings.py`:
   - `RAZORPAY_KEY_ID`
   - `RAZORPAY_KEY_SECRET`

## Gmail SMTP Setup

1. Enable 2-Factor Authentication on Gmail
2. Generate App Password: https://myaccount.google.com/apppasswords
3. Update settings.py:
```python
EMAIL_HOST_USER = 'your-email@gmail.com'
EMAIL_HOST_PASSWORD = 'your-16-digit-app-password'
```

## QR Scanning System Options

### Option 1: Web-Based Scanner (Recommended - Implemented)
**Technology:** HTML5-QRCode library
**Pros:**
- Works on any device with camera
- No additional app needed
- Fast and reliable
- Real-time feedback

**Implementation:** Already included in `scan_qr.html`

### Option 2: Mobile App Scanner
**Technology:** React Native with QR Scanner
**Pros:**
- Native performance
- Offline capability
- Better camera control

**Setup:**
```bash
npx react-native init EventScanner
npm install react-native-qrcode-scanner
```

### Option 3: Dedicated Scanner Device
**Hardware:** USB/Bluetooth QR Scanner
**Pros:**
- Fastest scanning
- Very reliable
- No camera needed

**Implementation:** Connect scanner to admin laptop, it types the QR data directly

### Option 4: Python Desktop App
**Technology:** OpenCV + PyQt5
**Pros:**
- Works offline
- Fast processing
- Custom UI

```python
pip install opencv-python pyzbar PyQt5
```

## Usage Flow

### For Users:
1. Browse events at `/`
2. Click "Book Now" on desired event
3. Fill booking form with validations
4. Complete Razorpay payment
5. Receive QR code via email
6. Show QR at venue entrance

### For Admins:
1. Access scanner at `/scan/`
2. Click "Start Scanner"
3. Point camera at user's QR code
4. System shows:
   - ✓ Green: Valid ticket, first-time check-in
   - ⚠ Yellow: Already checked in
   - ✗ Red: Invalid/not found
5. User details displayed on successful scan

## Troubleshooting

### Camera not working in scanner:
- Ensure HTTPS is enabled
- Grant camera permissions
- Try different browsers (Chrome recommended)

### QR code not generating:
- Check Pillow installation
- Verify media folder permissions
- Check qrcode library version

### Payment failing:
- Verify Razorpay keys
- Check test mode is enabled
- Inspect browser console for errors

### Email not sending:
- Verify Gmail App Password
- Check firewall settings
- Try with different email provider
